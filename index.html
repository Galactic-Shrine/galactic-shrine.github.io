Décision de départ

Choix retenus

  • Site entièrement statique pour un déploiement simple sur GitHub Pages.
  • Pas de build requis : tu peux pousser le dépôt tel quel.
  • Les standards restent en Markdown pour conserver leur source officielle.
  • Le design peut ensuite être remplacé par un thème plus avancé ou un générateur statique.
