# Galactic-Shrine Harmonized Documentation Standards

Cette édition harmonisée aligne l’ensemble des standards sur une même structure.

## Harmonisation appliquée

- même en-tête (`Version`, `Auteur / Author`, `Portée / Scope`) ;
- même structure en 12 sections ;
- même ordre bilingue `[FR]` puis `[EN]` ;
- mêmes rubriques transverses :
  - Objectif
  - Principes généraux
  - Format obligatoire des blocs
  - Convention bilingue
  - Éléments autorisés
  - Structure recommandée
  - Standard par type d’élément
  - Règles de style
  - Types à documenter en priorité
  - Exemple complet recommandé
  - Décision officielle recommandée
  - Résumé exécutable
- conservation des spécificités natives de chaque langage.

## Version

Version harmonisée : **1.2.0**
