Déploiement

Déploiement GitHub Pages

Le workflow Pages est inclus et le domaine personnalisé reste pris en charge.

Étapes

  1. Activer GitHub Actions dans Settings → Pages.
  2. Lancer npm install puis npm run build localement si besoin.
  3. Pousser sur main pour déclencher le déploiement.
  4. Conserver le fichier src/CNAME aligné avec le domaine configuré côté GitHub.
