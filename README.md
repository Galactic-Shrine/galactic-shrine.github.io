# galactic-shrine.github.io

Starter statique pour le portail GitHub Pages de Galactic-Shrine.

## Contenu

- Page d'accueil
- Page `Documentation Standards`
- Page `Coding Standards`
- Téléchargements ZIP
- Fichiers Markdown sources déjà intégrés
- `404.html`
- `.nojekyll`

## Déploiement rapide

1. Créer le dépôt `galactic-shrine.github.io`
2. Copier tout le contenu de ce dossier dans la racine du dépôt
3. Pousser sur `main`
4. Dans GitHub :
   - `Settings`
   - `Pages`
   - `Build and deployment`
   - Source : `Deploy from a branch`
   - Branch : `main`
   - Folder : `/ (root)`

## Domaine personnalisé

- Utiliser `CNAME.example` comme base si tu veux brancher `docs.galactic-shrine.com` ou `standards.galactic-shrine.com`.

## Évolutions possibles

- Ajouter une recherche
- Générer des pages HTML complètes à partir des Markdown
- Migrer vers Jekyll, MkDocs, Docusaurus ou Astro
- Ajouter guides, changelogs, roadmap, API docs
