# Galactic-Shrine Standards — Eleventy Migration

Cette version migre le portail `galactic-shrine.github.io` vers **Eleventy (11ty)**.

## Objectif

- conserver l’identité visuelle validée ;
- garder le déploiement sur **GitHub Pages** via **GitHub Actions** ;
- remplacer les pages HTML maintenues à la main par des **sources de contenu** et des **layouts** ;
- continuer à publier les standards au format **HTML** et **Markdown**.

## Prérequis

- Node.js 18 ou supérieur
- npm

## Commandes

```bash
npm install
npm run dev
npm run build
```
